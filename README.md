## 💳 Sistema de pagamento integrado com Mercado Pago Pix

Um dos principais diferenciais do projeto é a integração com a API do **Mercado Pago**, permitindo um fluxo automatizado de contratação e liberação de anúncios.

O sistema utiliza pagamentos via **Pix** para validar a publicação dos imóveis anunciados.




## 📌 Sobre o projeto

O **Busca Busca Arealva** é uma plataforma web desenvolvida para facilitar a divulgação e busca de imóveis na cidade de Arealva/SP.

O projeto foi criado utilizando **Inteligência Artificial como ferramenta de desenvolvimento**, demonstrando como novas tecnologias podem acelerar a criação de soluções digitais completas.

A plataforma permite que usuários cadastrem imóveis, adicionem informações, fotos e localização, disponibilizando anúncios de forma automatizada após a confirmação do pagamento.

## 🚀 Funcionalidades

### 👤 Usuários
- Cadastro de usuários
- Login e gerenciamento de conta
- Criação de anúncios de imóveis
- Edição e gerenciamento dos próprios anúncios

### 🏡 Anúncios
- Cadastro de imóveis para venda ou aluguel
- Inserção de descrição, valores e características
- Upload de imagens
- Seleção de localização utilizando Google Maps
- Exibição pública dos imóveis anunciados

### 💳 Pagamentos
- Integração com API do Mercado Pago
- Validação automática de pagamentos via Pix
- Liberação automática do anúncio após confirmação

### 📊 Administração
- Painel administrativo
- Visualização e gerenciamento dos anúncios
- Análise de dados da plataforma
